Controls Mapped

- Transport: Play, Stop, Record, Loop, Rewind, Fast Forward — all with LED feedback
- Navigation: Track Left/Right, Bank Prev/Next, Page switching via Screen Up/Down
- Pads (Plugin page): Preset Prev/Next, Undo/Redo, Plugin switching, Zoom, Navigate, Metronome, Quantize, Editor toggle, MixConsole toggle — all with color-coded LEDs
- EQ SubPages: Cycle Gain/Freq/Q with Right Up/Down buttons
- LCD Screens: Live parameter names, values, and knob position indicators on all 8 columns + center screen showing page info
- LED Feedback: Track colors reflected on strip LEDs, mute/solo/rec arm state shown on button LEDs, transport LEDs lit

To Test

1. Open Cubase Pro 15
2. Go to Studio > MIDI Remote (or lower zone MIDI Remote tab)
3. The script should auto-detect the SL MkIII on the InControl port
4. Press the InControl button on your SL MkIII
5. You should see the LCD screens initialize with "SL MkIII / Cubase 15"
6. Use Screen Up/Down buttons to cycle between the 4 pages
